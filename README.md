# Backend
Instalando o serve

`npm install`

Rodando deploy

`npm run deploy`

Rodando server

`npm run serve` 

Executando testes

`npm run test`

# Frontend

Instale as dependencias

`npm install`

Execute o angular

`ng serve`

Acesse o site no `localhost:4200`
